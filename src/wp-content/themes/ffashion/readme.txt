fFashion is a fully Responsive Fashion WordPress Theme.

== Description ==

fFashion is a fully Responsive Fashion WordPress Theme suitable for a fashion blog, fashion models website,
a site for a  fashion clothes, fashion fashion, etc. Features include Built-in Homepage (display Only for
a Static Front-Page), Social icons in header: Facebook, Twitter, LinkedIn, Instagram, RSS Feeds,
Tumblr, YouTube, Pinterest, Flickr, Vine, Multi-Level Drop-Down Header Menu, Admin Customize Options (Social
Sites, Slider Images and Content, Header Logo and Footer Copyright Text), Multiple Browser Compatible (IE9,
IE10, IE11, Chrome, Firefox, Opera, and Safari), FontAwesome Integration, Full-Width Page Template, Widget
Areas: Sidebar  Widget Area, 3 Homepage Widget Areas, 3 Footer Widget Areas, W3C Markup Validated, Search
Engine Optimized (SEO), Mobile-Friendly, and Translation Ready.
DEMO: https://tishonator.com/demo/ffashion


== Frequently Asked Questions ==

= How to Display the Homepage Slider =

1. Create a Static Front Page - the slider is visualized ONLY when there is a static front page set for your website. So, the first step is:

1.1. Login to your Admin Panel
1.2. Navigate to Left Menu -> Settings -> Reading
1.3. Select 'A static page (select below)', then choose pages for 'Homepage' and 'Posts page'
1.4. Save changes

Reference: https://codex.wordpress.org/Creating_a_Static_Front_Page

2. Update Slider Settings
2.1. Login to your Admin Panel
2.2. Navigate to Left Menu -> Appearance -> Customize
2.3. Open 'Slider' Section
2.4. Check the 'Display Slider' checkbox
2.5. Set Slider Images ('Slide # Image' fields) and Text ('Slide # Content' fields) for the different Slides
2.6. Save Changes
